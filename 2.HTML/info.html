Documentación

El DOCTYPE html sirve para crear un sitio html

El html es donde comienza nuestro sitio web, dónde empieza el código que lee el navegador

El atributo lang es para poner el idioma en o es, pero es para que google lo identifique

El title para el titulo de la cabecera de la página

El head es la cabecera, solo muestra información al navegador web y en algunos casos buscadores Google, Yahoo, etc.

El meta charset="UTF-8 es el tipo de codificación, para que el navegador nos coja cosas que no están en el alfabeto inglés Ej: la ñ, acebtos y caracteres especiales

El body que es el cuerpo de la página donde ponemos todo el contenido de nuestro sitio web
